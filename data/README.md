# Data Directory

This directory is intentionally empty. All data must be downloaded from WRDS.

## How to obtain data

1. Ensure you have a valid WRDS institutional subscription
2. Run: `python code/00_download_wrds_data.py`
3. The script will prompt for your WRDS credentials and download all required datasets

## Required datasets

| File | Source | WRDS Table |
|------|--------|-----------|
| `directors_officers_full.csv` | Audit Analytics | `audit.feed17_director_and_officer_changes` |
| `restatements_with_dates.csv` | Audit Analytics | `audit.feed39_financial_restatements` |
| `auditor_company.csv` | Audit Analytics | `audit.feed01_audit_opinions` |
| `compustat_financials.csv` | Compustat | `comp.funda` |

## Intermediate files (generated by scripts)

These files are created by the analysis scripts and do not need to be downloaded:

- `yearly_interlock_edges.csv` — Generated by `01_build_edge_lifecycles.py`
- `edge_lifecycles.csv` — Generated by `02_temporal_contagion.py`
- `temporal_contagion_pairs.csv` — Generated by `02_temporal_contagion.py`
- `integrated_panel.csv` — Generated by `03_causal_analysis.py`
